# my diet diary

## 소개 💁🏻‍♀️

- 게시판을 응용하여 만들었습니다.
- 그림과 함께 제가 먹은 음식과 정보를 기록 할 수 있습니다.

<br />

### 기능 🌟

1. 글 작성
2. 달력에 기록 된 사진 노출.
